# LingBlog - 一款以二次元，圆润风格为主的个人主页+博客

因为感觉博客模版太多，还是想自己开发一个来玩玩，于是就有了这个项目。目前还在施工中ing...

## 博客特点

- [x] 二次元风格
- [x] 主页，博客分离
- [x] 星空样式，青色主题
- [x] 卡片式博客展示
- [x] 个人项目，绘画作品展示

## 技术栈

- Vue3
- Vue Router
- Pinia
- Vitepress
- Tailwind Css
- TypeScript
- Vite
- ESLint
- Prettier

## 在线预览（博客部分）

[Github Pages](https://slimeboyowo.github.io/LingBlog/ling-chat/backend)

## 如何使用？

1. 克隆项目到本地

```sh
git clone https://github.com/SlimeBoyOwO/LingBlog.git
```

2. 安装依赖

```sh
pnpm install
```

3. 运行个人主页部分

```sh
pnpm dev
```

4. 运行博客部分

```sh
pnpm run docs:dev
```
