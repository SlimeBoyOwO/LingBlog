# 主要项目一览

## 1. LingChat - 智能情感对话类 Galgame 人工智能对话项目

### 简介

个人开发的一款灵动の人工智能聊天陪伴助手，使用深度学习预测 LLM 输出情绪，利用短句预测大幅提升预测准确度。结合 Vits 语音服务+前端舞台演出，打造沉浸式对话体验。

### 技术栈

- 前端：Vite + Vue3 + TypeScript + Pinia
- 后端：FastAPI + Python
- 语音：Style Bert Vits / Vits / SoVits
- 数据存储：SQLite
- 情绪识别：自训练 BERT 微调模型

## 2. QinKitPVP2 - Java Minecraft 服务器插件项目

### 简介

个人开发的用于胡萝卜之屯的 Minecraft 服务器小游戏插件，主要小游戏有：

1. 完全原创的职业战争小游戏，引入身法系统，天气系统，各种奇葩技能，完全全新的 PVP 战斗体验，支持全自定义职业内容。
2. 完全原创的 RTS 游戏“悦灵战争”，全原创防御塔和兵种，完全颠覆的游戏玩法，超大战场，超大游戏系统。
3. 在完全的 mc 服务器原版的 pvz mc 改版。全局植物防御，大量原创的僵尸，独特的 PVE 玩法。

### 技术栈

- 主要编程语言：Java
- 使用 API: Minecraft API, FastBoard API
- 使用框架：Spigot, Bukkit
- 数据存储：MySQL
